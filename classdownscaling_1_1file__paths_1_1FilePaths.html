Generative Additive Model Files

Group of attributes for generative additive model files.

The 'gam_diagnostics' file saves the pdf from the 'gam.check(res)' command, while the 'gam_summary' file saves the text output of all assessment commands and the 'gam_model' file saves the GAM model itself for reuse in the future. Once for minimum-mean discharge relationship, once for maximum-mean.

Sets the file paths for output files used in the analysis.

This method generates and assigns file paths for various output files based on the input function and the string representation of months.

Parameters
months(list of int) A list of integers representing the months used in the analysis.
function(str) The function to use for the downscaling, between:
  • "Singh2014": the original function
  • "Sigmoid_d": the glacial function
  • "Sigmoid": the simplified glacial function
