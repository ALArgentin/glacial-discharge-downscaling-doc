Compute a reference for the provided metric (goodness of fit) by block bootstrapping the observed series n_evals times (100 times by default), evaluating the bootstrapped series using the provided metric and computing the mean of the results.

Parameters
all_bootstrapped_FDCs_dfs(dataframe) All bootstrapped flow duration curves as different columns.
observed_FDCs_df(dataframe) The observed flow duration curves.
metric(str) The abbreviation of the function as defined in HydroErr (https://hydroerr.readthedocs.io/en/stable/list_of_metrics.html) Examples: nse, kge_2012, ...
Returns
The mean value of n_evals realization of the selected metric.
